# 📀 Správce CD sbírky
## Seznam změn:
* 20.03.2026 - Vytvoření frontendu, náčrt backendu, vytvoření vlastností (public string Nazev,..)
* 25.03.2026 - Zprovoznění kolekce, přidávacího a mazacího tlačítka

## Co bude spravovat?
Program bude sledovat sbírku mých CDs, které vlastním. Zároveň bude v aplikaci implementován wishlist, kde budou všechna CDs, které bych si chtěl jednoho dne zakoupit.<br>
Současně bude v aplikaci řazení CDs ve sbírce podle roku vydání a abecedně A-Z, jak název alba, tak název umělce.

## Jaká data bude sbírat?
* Název alba
* Umělec
* Datum vydání
* Seznam skladeb
* Stopáž
* Žánr
* Vydavatelství
* (přibližné) Datum přidání do osobní sbírky

## Technická část:
Aplikace je naprogamovaná v jazyce C# a využívá technologii WPF. Program je připraven na implementaci databáze.
